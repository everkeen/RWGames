      Settings

      Disables creature AI to improve performance; also removes survival stats.

      Disables special graphics like glow effects to improve performance.

      Makes explosions use 64 rays instead of 32 for a smoother appearance, at the cost of performance.

      Limits the number of particles that can exist at once to improve performance. If the limit is reached, new particles will not spawn until some existing ones despawn.

      Warning: Setting the particle limit too high may cause performance issues or crashes on some devices.

      Forces the game to use a specific renderer type. By default, the game will choose the best available renderer for the user's device.

      The WebGL renderer may not work on all devices and browsers; as WebGL2 is required, some users may experience graphical glitches or reduced performance. The old renderer is less performant and may not support all features, but should work on a wider range of devices.

      Optimizes special effects like glow and particle effects for better performance; Game may look less visually appealing.

      Controls

      Desktop

      • Left Click: Use tool
      • Right Click: Use tool (alternative)
      • P: Pause/Unpause
      • D: Toggle debug menu (above game)
      • Minus (-): Decrease brush size
      • Equal (=): Increase brush size
      • C: Clear with terrain
      • Shift + C: Clear without terrain
      • Shift: Hold to intensify tool
      • S: Open settings

      Mobile

      • Tap and hold: Use tool
      • All other controls are available through the mobile interface
